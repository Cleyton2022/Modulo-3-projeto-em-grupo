# Terceiro projeto em grupo

Podem ser utilizados os métodos Get, Put, Post e Delete na rota/sugestões.
Para iniciar
Ultilizar o comando no cmd ou pw shell "npm install"
Após usar "npm run dev"

## Dependências utilizadas
- Axios;
- Bootstrap;
- Framer Motion;
- React;
- React DOM; 
- React Router DOM;
- React Icons.
